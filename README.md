## 🔧 Troubleshooting

### Files do not open

- Make sure the download finished
- Check that the file is not still inside the ZIP archive
- Confirm that Cinema 4D is installed
- Try opening the file from the extracted folder

### Textures are missing

- Check the texture folder
- Keep file names unchanged
- Use the search or relink tool in Cinema 4D
- Make sure the project and textures stay in the same parent folder

### Scripts do not run

- Confirm that the script is in the right folder
- Check that your Cinema 4D version supports the script
- Open one script at a time
- Restart Cinema 4D if the tool panel does not refresh

### The app asks for access

- Right-click the file and choose Run as administrator if needed
- Allow access only if you trust the file and source
- Keep the files in a local folder with normal read and write access
